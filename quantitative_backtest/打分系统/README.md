# 股票打分系统

## 简介
专注于股票基本面评分的系统，通过计算多个财务因子对股票进行综合评分和排名。

## 使用方法

1. **修改参数**：编辑 `run_scoring.py` 中的日期和输出数量
2. **运行系统**：`python run_scoring.py`
3. **查看结果**：在 `output/scores/` 目录下找到Excel文件

## 文件结构

- `config/` - 配置文件
- `src/core/` - 核心计算模块
- `src/utils/` - 工具函数
- `output/scores/` - 评分结果输出
- `run_scoring.py` - 主程序

## 系统特点

- 🎯 专注打分功能，简单易用
- 📊 支持多因子综合评分
- 📁 直接输出Excel格式结果
- 🔧 参数化配置，灵活调整
